3号地块「配套境」

慢享北里·2号地块专属配套核心·研学刚需配套服务中心

首页 六境分区 3号地块「配套境」

核心定位

作为 2 号地块（研学境）专属配套核心，聚焦 "研学刚需配套 + 轻社交延伸"，承接 2 号研学团队研讨、工具使用、课后休憩需求，衔接 4 号（文艺境）、6 号（生活境）形成全域联动，打造 "刚需适配、轻运营、高衔接" 的配套服务空间，保障研学体验连贯、主客共享便捷。

核心信息

  • 地块编号： 3号地块
  • 核心定位： 2号地块专属配套核心
  • 核心业态： 研学研讨室+工具驿站+文化茶座
  • 核心客群： 研学团队、本地居民、游客
  • 核心特质： 刚需适配·轻运营·高衔接

核心业态落地细则

模块1：研学研讨室（核心业态）

空间布局

  • • 1间主研讨室（30-40㎡，40人容量）
  • • 2间小型研讨室（15-20㎡，8-10人/间）
  • • 配可移动桌椅、多媒体设备、隔音墙面
  • • 预留成果展示区，做防滑、防撞、无障碍适配

运营核心

  • • 研学团队提前1天预约
  • • 优先保障工作日9:00-16:00研学使用
  • • 非高峰时段对本地居民开放
  • • 配备1名管理员，负责设备调试、巡查与场地复位

收费适配

  • • 主研讨室：200元/小时
  • • 小型研讨室：80元/小时
  • • 2号合作研学机构享8折
  • • 本地居民：10元/人/次（限小型研讨室非高峰）

模块2：工具驿站（刚需配套）

空间布局

  • • 15-20㎡专属空间
  • • 分储物、取用、维修区
  • • 配多层货架、维修台、监控、消防设施
  • • 按 "研学工具、维修耗材、应急物资" 分类存放

运营核心

  • • 实行 "入库-领用-归还-维护" 闭环
  • • 每日盘点账物一致
  • • 尖锐工具单独存放、专人监管
  • • 未成年人禁止单独领用

收费适配

  • • 常用工具（卷尺、记号笔）：研学团队免费领用
  • • 尖锐/专用工具：5元/件/次
  • • 合作团队可办月卡（200元/月）
  • • 零散客群：2-10元/件/次（需缴押金）

模块3：文化茶座（延伸业态）

空间布局

  • • 20-25㎡空间
  • • 分安静品茗区（适配研学休憩）
  • • 轻社交区
  • • 配茶桌、茶具、冷藏柜、消毒设备

运营核心

  • • 研学团队设快速点单通道
  • • 每日做好场地、茶具清洁消毒
  • • 茶具热力消毒≥85℃、≥10分钟
  • • 搭建社群运营，提升客群留存

收费适配

  • • 茶品：15-35元/杯
  • • 茶点：8-20元/份
  • • 研学团队享团体套餐优惠（满10人9折）
  • • 提供传统+年轻化茶品+轻茶点（标注过敏源）

落地注意事项

建筑适配

遵循历史建筑保护要求，不破坏结构，做好隔音、防滑改造，确保空间安全舒适。

动线衔接

与2号地块无缝对接（步行≤2分钟），通过2-4号过渡广场联动4号地块，形成流畅的空间动线。

合规保障

符合研学基地设施规范，消防、应急、卫生达标，工具驿站、茶座完成经营备案。
